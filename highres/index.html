About

This 3d Model was made in Open Drone Map deployed on a virtual machine (VM) on Google Cloud. The orginal file size was ~10 gigabytes and contained 709 photos. Processing in Google Cloud took 3 hours on a VM with 128g memory and 16 cpu. This model was processed into a 3d interactive model in QGIS. The model displayed here is a subset of the larger area and covers approximately 1% of the total area processed.
